## ğŸ“ Image Preparation Steps

### Step 1: Download Images
1. Go to Unsplash/Pexels
2. Search for your dish/scene
3. Download highest quality
4. Save with descriptive name

### Step 2: Optimize Images

**Using TinyPNG:**
1. Go to tinypng.com
2. Upload your image
3. Download compressed version
4. Can compress up to 20 at once

**Using Squoosh:**
1. Open squoosh.app
2. Drop your image
3. Choose format (JPG/PNG)
4. Adjust quality (70-80% is good)
5. Download

### Step 3: Rename Files
Use exact names from the list above:
```
burger.jpg (not Burger.JPG or burger_image.jpg)
biryani.jpg (not chicken-biryani.jpg)
```

### Step 4: Place in Folder
Move all images to:
```
hot-spot-kitchen/static/images/
```
